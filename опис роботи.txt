Ідея роботи

Метою проєкту є створення Telegram-бота для притулку для тварин. Планується:
* Розробити Telegram-бота, який допоможе покращити комунікацію між притулком і потенційними відвідувачами або волонтерами.
* Домовитися про співпрацю з реальним притулком, щоб адаптувати функціонал бота до їхніх потреб.
* Представити результати роботи в навчальному закладі.
* Після завершення — передати бота притулку для подальшого використання.

 Опис реалізації

1. **Створення бота**
   * За допомогою сервісу BotFather у Telegram було створено нового бота, сформовано його початкову оболонку та отримано унікальний токен доступу.
2. **Розробка**
   * У середовищі Visual Studio Code (далі — VS Code) було встановлено:

     * Мову програмування Python.
     * Бібліотеку **python-telegram-bot**, яка дозволяє легко реалізовувати функціонал Telegram-ботів.

3. **Перший примірник**
   * Реалізовано базові команди:
     * `/start` — запуск бота.
     * `/reset` — перезапуск сесії.
   * Додано кнопки:
     * Три базові кнопки (наразі не мають функціоналу — «пустишки»).
     * Окрема кнопка для виклику `/reset`, щоб користувачам не потрібно було вводити її вручну.
   * Згенеровано тимчасовий аватар для бота.
   * Проведено початкові тести роботи бота.
